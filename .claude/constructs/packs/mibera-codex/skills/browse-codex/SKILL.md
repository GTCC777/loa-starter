# Browse Codex

Navigate the Mibera Codex by dimension — find Miberas grouped by archetype, ancestor, drug, element, era, tarot, or swag rank.

## Browse Dimensions

All browse indices live in `browse/`:

| Dimension | File | What it groups |
|-----------|------|----------------|
| Archetype | `browse/by-archetype.md` | 4 archetypes (Freetekno, Milady, Chicago Detroit, Acidhouse) |
| Ancestor | `browse/by-ancestor.md` | 33 cultural lineages |
| Drug | `browse/by-drug.md` | 78 molecules |
| Element | `browse/by-element.md` | 4 elements (Fire, Water, Air, Earth) |
| Era | `browse/by-era.md` | 11 birthday eras (13,166 BCE – 2024 CE) |
| Swag Rank | `browse/by-swag-rank.md` | SSS through F ranks |
| Tarot | `browse/by-tarot.md` | 78 tarot cards |
| Grails | `browse/grails.md` | 42 hand-drawn 1/1s |

## How to browse

1. Pick a dimension from the table above
2. Read the corresponding `browse/by-{dimension}.md` file
3. Each section lists Mibera IDs linked to their individual files at `miberas/{NNNN}.md`
4. Cross-dimensional breakdowns (e.g. archetype distribution within each ancestor) are inline in the enriched browse pages

## Entry point

Start at `browse/README.md` for a human-readable overview of all dimensions.
