# Cypherpunk Floppies

**Type:** 1/1 Items

12 unique held items, each a floppy disk featuring a pioneer of the cypherpunk movement. Each appears on exactly one Mibera. There is also a generic "Cypherpunk" floppy item (swag score 4) that appears on 41 tokens.

Each named floppy is a 1-of-1 item with swag score 5 (max).

## Generic Cypherpunk Item

The generic [Cypherpunk](../../traits/items/general-items/cypherpunk.md) floppy (swag score 4) appears on **41 tokens** in the collection. It features a sheet of cryptographic-looking vertical strokes — a visual reference to encrypted data on a floppy disk.
