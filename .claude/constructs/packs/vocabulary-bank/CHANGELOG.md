## [0.1.0] - 2026-03-13

### Added
- Initial construct release
- Anchor-Space-Gravity philosophy document (universal, product-agnostic)
- Resolution rules (8 vocabulary rules + 5 lore-in-operational-comms rules)
- Vocabulary bank template (tiered lexicon: T1 chain-standard, T2 earned, T3 reserved)
- Channel registry template (channel-to-register mapping with tonal gap rule)
- `/synthesize-vocabulary` skill — extract terms from codebase, classify into tiers
- `/audit-vocabulary` skill — lint copy against the vocabulary bank, report violations
- Identity layer (Lexicographer archetype, structural thinking style)
- Install script with idempotent grimoire setup

### Research
- 70+ sources across 8 ecosystems (Riot, Bungie, Square Enix, CCP, Valve, Supergiant, Yuga Labs, DeFi)
- Channel-specific voice registers: no studio uses explicit channel voice guides — register is emergent
- Lore in operational comms: world vocabulary works when it IS the technical identifier, not when it replaces one
- Vocabulary governance at scale: federated model (shared schema, per-product autonomy) outperforms centralized
- FFXIV localization pipeline as the deepest vocabulary governance system found in any studio
