# Visual Semiotics

What do shapes, materials, and motion mean to human viewers? This skill grounds visual decisions in cognitive science — embodied cognition, Gestalt principles, and universal shape semantics.

## Core Frameworks

### Embodied Metaphor (Lakoff & Johnson)
- MORE IS UP — height/verticality = magnitude, growth, power
- CONTAINER metaphor — enclosure = protection, security, belonging
- PATH metaphor — trajectories = progress, journey, direction
- FORCE metaphor — gravity, attraction = commitment, pull

### Universal Shape Semantics
- Circles/rings → protection, unity, completeness, cycles
- Vertical bars → magnitude, comparison, hierarchy
- Horizontal lines → stability, grounding, horizon
- Angular/sharp → precision, cutting-edge, danger
- Organic/flowing → natural, growth, life
- Dense clusters → community, network, scale

### Motion-as-Meaning
- Breathing/pulsing → alive, present, organic
- Orbiting → system, structure, gravitational pull
- Rising → growth, progress, accumulation
- Converging → community, attraction, consensus
- Expanding → reach, influence, broadcast
